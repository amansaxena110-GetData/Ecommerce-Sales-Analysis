# 🛒 E-Commerce Sales Analysis Dashboard

![Dashboard](Screenshot%202026-05-07%20104500.png)

## 📌 Project Overview
This project analyzes a Brazilian E-Commerce dataset from Kaggle (Olist Store) 
with nearly 100,000 orders. The goal is to extract business insights about 
revenue, customer behavior, product performance, and delivery efficiency 
using Python, SQL, Excel, and Power BI.

---

## 🛠️ Tools & Technologies Used

| Tool | Purpose |
|---|---|
| 🐍 Python (Pandas, Matplotlib, Seaborn) | Data cleaning & visualization |
| 🗄️ SQL Server (SSMS) | Data querying & analysis |
| 📊 Microsoft Excel | Pivot tables & reporting |
| 📈 Power BI | Interactive dashboard |
| 🐙 GitHub | Version control & portfolio |

---

## 📁 Dataset
- **Source:** [Brazilian E-Commerce Dataset - Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- **Size:** 9 CSV files, ~100,000 records
- **Tables:** Orders, Customers, Products, Sellers, Payments, Reviews, Order Items

---

## ❓ Business Questions Answered
1. What is the total revenue generated?
2. Which months have the highest orders and revenue?
3. Which product categories generate the most revenue?
4. What is the most preferred payment method?
5. What is the average customer review score?
6. What percentage of orders are delivered successfully?
7. Which sellers generate the highest revenue?
8. What is the average delivery time?

---

## 📊 Key Insights Found

- 💰 **Total Revenue:** R$ 13.59 Million
- 📦 **Total Orders:** 99,441
- ⭐ **Average Review Score:** 4.09 out of 5
- 🚚 **Delivery Success Rate:** 97% orders delivered successfully
- 📅 **Peak Month:** August has the highest number of orders
- 🏆 **Top Category:** Health & Beauty with R$ 1.25M revenue
- 💳 **Payment Preference:** 74% customers prefer Credit Card
- ⏱️ **Average Delivery Time:** 12 days
